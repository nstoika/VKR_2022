Загрузить на локальный компьютер файлы из репозитория, включая файл с датасетом data_norm.csv
Выложить файл в корневой каталог на Google Drive
Открыть Google Colab, подключить Google Drive.
Проверить работоспособность раздела «Загрузка данных», при необходимости поправить ссылку. 
Открыть Visual Studio Code. 
Создать виртуальное окружение. 
–	Версия Python  3.9.12
–	версия conda 4.13.0
Активировать виртуальное окружение через терминал командной строки. 
Установить библиотеки flask, pickle-mixin, sklearn, numpy, pandas, gunicorn.
Загрузить в виртуальное окружение файлы app.py и main.html (в папку template1, папку создать в корне)
Запустить приложение командой python3 app.py, для использования перейти в localhost http://127.0.0.1:5000.
