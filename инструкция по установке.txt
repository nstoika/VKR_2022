1.	Загрузить на локальный компьютер файлы из репозитория, включая файл с датасетом data_norm.csv
2.	Выложить файл в корневой каталог на Google Drive
3.	Открыть Google Colab, подключить Google Drive.
4.	Проверить работоспособность раздела «Загрузка данных», при необходимости поправить ссылку. 
5.	Открыть Visual Studio Code. 
6.	Создать виртуальное окружение. 
–	Версия Python  3.9.12
–	версия conda 4.13.0
7.	Активировать виртуальное окружение через терминал командной строки. 
8.	Установить библиотеки flask, pickle-mixin, sklearn, numpy, pandas, gunicorn.
9.	Загрузить в виртуальное окружение файлы app.py и main.html (в папку template1, папку создать в корне)
10.	Запустить приложение командой python3 app.py, для использования перейти в localhost http://127.0.0.1:5000.
